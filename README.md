# vuln-warehouse
مخزن تخصصی برای انتشار تحلیل‌های امنیتی، کدهای اثبات مفهوم و راهکارهای مقابله با آسیب‌پذیری‌های کشف‌شده در نرم‌افزارهای مختلف.
هدف اصلی، افزایش آگاهی امنیتی در جامعه توسعه‌دهندگان و پژوهشگران حوزه امنیت سایبری از طریق ارائه مستندات فنی دقیق و قابل‌بازتولید است. تمامی محتوا صرفاً برای اهداف آموزشی و پژوهشی منتشر می‌شود.
